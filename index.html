Первые проекты на Python для начинающих

Проект 1: Парсер сайтов

Научитесь собирать данные с веб-страниц автоматически. Востребованный навык для аналитики.

# Основы Python: парсинг веб-страниц
import requests
from bs4 import BeautifulSoup

url = "https://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')
title = soup.find('title').text
print(f"Заголовок страницы: {title}")
